# HTML 面试题

## 如何理解 HTML 语义化？

* **让人更容易读懂（增加代码的可读性）**
* **让搜索引擎更容易读懂（SEO）**

## 默认情况下，哪些 HTML 标签是块级元素，哪些是内联元素？

#### display: block/table;   **有div  h1  h2  table  ul  ol  p 等**

#### 不管里面内容有多少，独占一行

#### display: inline/inline-block;   **有 sapn  img  input  button 等**

#### 不会独占一行，只会挨着往后排，直到浏览器的边缘换行为止
